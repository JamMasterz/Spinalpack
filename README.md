# Spinalpack
This is the base plugin for the Spinalcraft Suite. It facilitates database operations for the other Spinalcraft plugins and provides basic informational commands for players.

-

####Spinalcraft Suite
The Spinalcraft Suite is a collection of Minecraft plugins developed primarily for operation on the [Spinalcraft server](http://reddit.com/r/spinalcraft).
